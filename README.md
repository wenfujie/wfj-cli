# 个人脚手架

### 使用

```bash
# 全局安装
npm i @wenfujie/cli -g

# 初始化项目
wfj create projectName
```

### 更新脚手架

1. 修改 package.json 下的 version
2. 运行指令

   ```bash
   npm run publish
   ```
